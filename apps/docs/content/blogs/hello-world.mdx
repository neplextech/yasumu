# Hello World 👋

Welcome to the official Yasumu blog! We're thrilled to have you here
as we embark on this exciting journey to revolutionize how developers
work with APIs.

## What is Yasumu?

Yasumu is a modern, open-source **API Laboratory** designed to help
developers design, test, and debug their API workflows with
confidence. Built with cutting-edge technologies like Tauri and a
custom JavaScript runtime, Yasumu provides a powerful yet lightweight
experience.

<Callout title="Open Source" type="info">
  Yasumu is completely open source and free to use. Check out our [GitHub
  repository](https://github.com/neplextech/yasumu) to contribute or star the
  project!
</Callout>

## Key Features

Here's what makes Yasumu special:

- **🚀 REST API Testing** - Comprehensive HTTP request builder with
  full support for headers, body, authentication, and more
- **📧 SMTP Server** - Built-in catch-all email server for testing
  email workflows locally
- **⚡ Echo Server** - Integrated HTTP echo server for debugging and
  testing requests
- **📝 Scripting Support** - Email, Pre-request and post-response
  scripting with JavaScript or TypeScript
- **🔐 Environment Management** - Manage variables and secrets across
  different contexts
- **💾 Workspace Organization** - Keep your API definitions organized
  in workspaces
