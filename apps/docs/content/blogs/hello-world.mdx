## What is Yasumu?

Yasumu is a modern, open-source **API Laboratory** designed to help
developers design, test, and debug their API workflows with
confidence. Built with cutting-edge technologies like Tauri and a
custom JavaScript runtime, Yasumu provides a powerful yet lightweight
experience.

<Callout title="Open Source" type="info">
  Yasumu is completely open source and free to use. Check out our
  [GitHub repository](https://github.com/neplextech/yasumu) to
  contribute or star the project!
</Callout>
